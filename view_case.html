PKQ UNJA,Website System Program Khatam Qur'an

System PKQ UNJA


Project Campus

Years : 2023


The Problem


The PKQ system is a place where you can take advantage of the Al-Quran Khatam System to achieve spiritual achievements. using internet facilities that can be used by Jambi University students who live far from the Jami' Assalam Mosque, Jambi University, to make certificates after completing the process of reciting the Koran to completing the Al-Qur'an at the Jami' Assalam Mosque, Jambi University.


User & Audience


For this website to work, there are 2 sides to the website. And each website has its own users:

1. Users. User is an active student at Jambi University.

2. Administrator. The PKQ administrator or committee will use the website dashboard to manage participant data, input letters, confirm data.
